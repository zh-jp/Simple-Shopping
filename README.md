# Simple-Shopping
基于Vue2的移动端商城前端页面

## 后端数据接口
该项目仅作为`Vue2`实战练习，后端数据由黑马教程提供：

开发实测发现部分接口实际返回相应与接口文档不同，[详见此处](./error_in_document.md)。可根据教程视频修改调整，[视频链接](https://www.bilibili.com/video/BV1HV4y1a7n4)。
